### Maze Generation Algorithms:
- Randomized Prim's
- Recursive Division
- Wilson's Algorithm
- Fractal Tessellation
- Randomized Kruskal's
- Randomized DFS

### Maze Solving Algorithms:
- A* Search
- Dijkstra's Algorithm
- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- Greedy Best-First Search
